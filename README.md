# Rainfall Prediction Using Machine Learning

## Project Overview

Rainfall Prediction Using Machine Learning is a web-based application developed to predict rainfall using historical weather data and machine learning algorithms. The system analyzes various meteorological parameters such as temperature, humidity, pressure, wind speed, evaporation, and rainfall history to provide accurate rainfall predictions.

The project aims to help farmers, disaster management authorities, and the general public by providing reliable weather forecasting for better planning and decision-making.

---

# Objectives

* Predict rainfall accurately using machine learning techniques
* Analyze historical weather datasets
* Develop a responsive and user-friendly web application
* Improve decision-making in agriculture and weather monitoring
* Compare multiple machine learning algorithms for better accuracy

# Algorithms Used

## XGBoost

Main algorithm used for prediction due to:

* High accuracy
* Fast processing
* Better handling of large datasets
* Reduced overfitting

## Random Forest

Used for ensemble learning and comparison.

## Logistic Regression

Used as a baseline classification model.

# How to Run the Project

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/rainfall-prediction-ml.git
```

---

## 2. Navigate to Project Folder

```bash
cd rainfall-prediction-ml
```

---

## 3. Install Required Libraries

```bash
pip install -r requirements.txt
```

---

## 4. Run the Flask Application

```bash
python app.py
```

---

## 5. Open in Browser

```bash
http://127.0.0.1:5000
```
